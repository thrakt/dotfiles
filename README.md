# dotfiles

# install

* change shell to zsh
* install homebrew
* `brew install zplug`
* `brew install python`
* `python -m pip install pip`
* `pip install powerline-status`
* `brew install peco`
* `brew install reattach-to-user-namespace`
* install tmux
* install rbenv
* install mux
* install tpm

# update

* run `makelink.sh`
